# satellite_mission_control

## 📌 Overview

The **Satellite Mission Control Dashboard** is a modern web-based application designed to simulate and monitor satellite operations in real-time. It provides an interactive interface for tracking telemetry data, mission status, system health, and operational insights, similar to real-world space mission control systems.

---

## ✨ Features

* 📡 **Real-time Telemetry Monitoring**
  Track satellite parameters such as power, temperature, signal strength, and fuel levels.

* 📊 **Interactive Data Visualization**
  Dynamic charts and graphs for better understanding of system performance.

* 🛰️ **Mission Management**
  View and manage multiple satellite missions and their objectives.

* ⚠️ **Alerts & System Status**
  Get notified about anomalies, risks, and system health issues.

* 🧠 **AI/Smart Insights (Simulated)**
  Intelligent indicators for decision-making and system analysis.

* 🎛️ **Modern UI Dashboard**
  Clean and responsive interface built with reusable components.

---

## 🛠️ Tech Stack

* **Frontend:** Next.js (React)
* **Styling:** Tailwind CSS
* **UI Components:** Radix UI
* **Charts & Visualization:** Recharts
* **Icons:** Lucide React
* **Form Handling & Validation:** React Hook Form + Zod

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/satellite-mission-control.git
```

2. Navigate to the project folder:

```
cd satellite-mission-control
```

3. Install dependencies:

```
npm install
```

or

```
pnpm install
```

4. Run the development server:

```
npm run dev
```

5. Open in browser:

```
http://localhost:3000
```
